## Using library in your plugin

### Maven
```xml
<!-- Repository -->
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```

```xml
<!-- Dependency-->
<dependencies>
    <dependency>
        <groupId>com.github.thelblack</groupId>
        <artifactId>Cinventory</artifactId>
        <version>1.0.0</version>
    </dependency>
</dependencies>
```
